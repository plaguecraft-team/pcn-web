The PlagueCraft Network

The PlagueCraft Network is a server owned by Alex Frantz and co-owned by vsynkz.
This server runs on Spigot and hosts minigames such as multiple arcade games, duels, KitPVP, Factions, and more.

TheBoysBot - Discord

This bot is a private bot developed by Alex Frantz. It is used in a private server, with many commands such as setting game codes for game nights, movie nights, etc.
